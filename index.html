About
About the project

This website heavily utilizes Bootstrap v5.2 in addition to standard HTML5 / CSS3 for clean, professional, and responsive web UI/UX. Each page (four in total) contains different Bootstrap components and utilities, listed below:

  • General
    • Navbar
    • Footer
    • Grid Organization (rows/columns)
  • Home
    • Nav (tabs)
    • Cards
  • About Me
    • Image
  • Project History
    • Accordion - can collapse the color legend
    • Table
    • Badges
  • Contact Me
    • Form
    • Tooltips - hover over * by form element labels
    • Modal - displayed between form validation and submission
    • Alert - displayed after successful form submission
For detailed information on the project, view README.md.
Pages
Project pages

This project consists of the following four pages:

Home
Automatically opened when the website is launched. Contains high-level overviews of the project, including how it is organized and the technologies that were used. Accessible via the CGI and Home links in the navbar.
About Me
Contains various information about myself as well as a responsive image. Accessible via the About Me link in the navbar.
Project History
Contains a table of information on projects I have worked on since joining CGI, including technologies used, responsibilities, and a color legend. Accessible via the Project History link in the navbar.
Contact Me
Contains a mock contact form with client-side input validation, a confirmation modal, and alert on successful form submission. Accessible via the Contact Me link in the navbar.
For detailed information on each page, view README.md.
